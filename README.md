# PyFGH
PyFGH is a Python library for dealing with FGH

## Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install PyFGH.

```bash
pip install PyFGH
```

## Usage

```python
import PyFGH

# returns 'words'
basicmathmas.addition.addition_by_three(5)
```
